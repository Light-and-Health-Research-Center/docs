## Overview

To get started using the CS Calculator 2.0, head over to [https://cscalc.light-health.org](https://cscalc.light-health.org).

A calculation can be performed with the CS Calculator 2.0 in **3** easy steps.

1. <DocLink href="#selecting-sources">Select Sources</DocLink>
2. <DocLink href="#editing-input-variables">Edit Variables</DocLink>
3. <DocLink href="#viewing-results">View Results</DocLink>

## Selecting sources

The first step to perform a calculation is to select one or more sources from the **Source List** panel.

1. Locate the **Source List** panel.

   <Img
     src="/img/docs/cscalc/usage/getting-started/source-list.png"
     alt="source list"
     width={856}
     height={443}
   />

2. Click on a source to open a modal with more information about that source.

   <Img
     src="/img/docs/cscalc/usage/getting-started/add-source-modal.png"
     alt="source list"
     width={808}
     height={785}
   />

3. Click the **Add Source** button to add the source to the **Selected Sources** panel.
   <Img
     src="/img/docs/cscalc/usage/getting-started/selected-sources.png"
     alt="source list"
     width={857}
     height={145}
   />

For information on how to filter the Source List panel, see <DocLink href="/cscalc/usage/general-usage">General usage</DocLink>.

For information on how to add custom sources, download custom sources as JSON, and upload custom sources from JSON, see <DocLink href="/cscalc/usage/advanced-usage">Advanced usage</DocLink>.

## Editing input variables

After adding the source(s), you must specify the **illuminance(s)**, and adjust **Input Variables**.

1. Navigate to the **Calculations** tab by clicking the tab at the top of the calculator.

   <Img
     src="/img/docs/cscalc/usage/getting-started/calculations-tab.jpg"
     alt="calculations tab"
     width={1869}
     height={299}
   />

2. Locate the **Illuminance** input field in the **Selected Sources** panel.

   <Img
     src="/img/docs/cscalc/usage/getting-started/illuminance-input.jpg"
     alt="illuminance input"
     width={845}
     height={185}
   />

3. Enter in an **Illuminance** value into the input field and press _Enter_.

   <Img
     src="/img/docs/cscalc/usage/getting-started/illuminance-added.jpg"
     alt="illuminance input filled in with 200 lx"
     width={850}
     height={181}
   />

4. Adjust input variables in the **Input Variables** panel.
   <Img
     src="/img/docs/cscalc/usage/getting-started/input-variables.png"
     alt="input vaariables panel"
     width={850}
     height={333}
   />

For information on how to view source information and remove sources, see <DocLink href="/cscalc/usage/general-usage">General usage</DocLink>.

For background information about the pre-loaded sources and input variables, see <DocLink href="/cscalc/inner-workings">Inner workings</DocLink>.
