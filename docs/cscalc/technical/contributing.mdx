## Bug Reports and Feature Requests

If you'd like to submit a bug or request a feature for the CS Calculator 2.0, you can [create an issue on Github](https://github.com/Light-and-Health-Research-Center/CS-Calculator/issues/new) or [email us](mailto:victoria.lopez@mountsinai.org).

If submitting a bug report, please be as specific as possible regarding the browser and operating system of your machine, as well as the steps that can be taken to reproduce the bug.

For more information on Github issue, see [creating an issue](https://docs.github.com/en/issues/tracking-your-work-with-issues/creating-an-issue).

## Code contributions

If you'd like to contribute code to the CS Calculator 2.0, please create a pull request to our [Github repository](https://github.com/Light-and-Health-Research-Center/CS-Calculator).

If you're unfamiliar with Github, and want to learn how to contribute to open source projects, see [cloning a repo](https://docs.github.com/en/github/creating-cloning-and-archiving-repositories/cloning-a-repository-from-github/cloning-a-repository), [committing changes to your project](https://docs.github.com/en/github/committing-changes-to-your-project), and [proposing changes to your work with pull requests](https://docs.github.com/en/github/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests).
